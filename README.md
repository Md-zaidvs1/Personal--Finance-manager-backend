# SpendSense - Personal Finance Manager

A full-stack personal finance manager app built with MERN Stack.

## Tech Stack
- MongoDB, Express.js, React, Node.js
- Tailwind CSS, Chart.js, pdfkit, json2csv

## Features
- Expense tracking with category, date, description
- Recurring expenses
- Budget creation and monitoring with alerts
- Financial goal setting and progress tracking
- Income tracking
- Financial reports with charts
- CSV and PDF export
- Responsive design

## How to Run

### Backend
cd backend
npm install
node server.js

### Frontend
cd personal-finance-manager/frontend
npm install
npm run dev
